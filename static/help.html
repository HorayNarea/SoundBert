SoundBert - A simple REST based soundboard in Go.

List all available snippets

GET /list

Sample response:
{ "foo": "foo.mp3", "bar/baz": "bar/baz.wav" }

Play a single snippet

POST /play

Post data example:
filename=foo.mp3

Sample response:
{"playing":"foo.mp3"}

Example with path:
filename=bar/baz.wav

Response:
{"playing":"bar/baz.wav"}

Stop playback

GET /stop

Reload sounds

GET /reload_sounds
